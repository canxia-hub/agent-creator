# Agent Creator - 档位判定卡（v1）

> 用途：帮助在创建 Agent 时快速判断应使用 T1 / T2 / T3 / T4 哪个模板档位。

---

## 一、快速判定问题

依次回答以下问题：

1. 这个 Agent 是探索性的，还是要长期常驻？
2. 它只是轻量辅助，还是承担正式职责？
3. 它是否需要明确情景特化？
4. 它是否需要行业深度与交付能力？
5. 它是否承担管理、审查、架构、协调、核心决策职责？

---

## 二、判定规则

### T1：空白母板档
适用于：
- 仍在探索定位
- 高度手工定制
- 暂不追求完整成熟结构

### T2：基础起草档
适用于：
- 需要快速形成六核心文件骨架
- 但暂不进入正式常驻标准
- 轻量职能或试运行 Agent

### T3：标准规范档
适用于：
- 正式常驻 Agent
- 有明确情景分支
- 需要稳定结构与一致性
- 需要一定深度，但不是最高复杂核心

### T4：增强实战档
适用于：
- 主 Agent
- 管理 / 审查 / 架构 / 协调核心 Agent
- 需要深度设计与长期复杂任务能力
- 需要进入完整深度创建流程

---

## 三、简易评分法

每满足一项加 1 分：
- 长期常驻
- 明确领域职责
- 需要复杂多步任务处理
- 需要高质量交付能力
- 需要行业深度
- 需要管理/审查/协调能力

### 评分建议
- 0-1 分：T1
- 2 分：T2
- 3-4 分：T3
- 5-6 分：T4
